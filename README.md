## Prerequisites
- Docker
- Docker Compose

## Getting Started
1. Clone the repository:
    ```sh
    git clone https://github.com/mikelovato/Mona.git
    cd Mona
    ```

2. Run the project with Docker Compose:
    ```sh
    docker-compose up --build
    ```
